üìã Deskripsi Ujian

Materi Ujian: Bootstrap 5 - Styling dan Layout

Siswa diminta mengedit file HTML yang sudah disediakan dengan cara menambahkan class-class Bootstrap 5 untuk memperindah tampilan, menata layout, dan menyesuaikan UI sesuai dengan kreativitas masing-masing.

  • Struktur HTML tidak boleh dihapus
  • Siswa boleh menambahkan atribut class Bootstrap
  • Siswa boleh menambahkan div wrapper tambahan bila diperlukan
  • Gunakan Bootstrap utility classes untuk styling
  • Fokus pada responsive design dan estetika
